# EquiposDeFutbol
Prueba tecnica con springboot

Usuario de prueba para login:
{
  "username": "test",
  "password": "12345"
}
al endpoint http://localhost:8080/auth/login

Formato para crear un nuevo usuario:
{
  "username":"username",
  "password":"12345",
  "nombre":"nombre",
  "apellido":"apellido",
  "pais":"pais"
}
al endpoint http://localhost:8080/auth/register
